# SIMKOM

Sistem Informasi Komunikasi (SIMKOM) adalah project berbasis web yang dikembangkan untuk mendukung proses komunikasi, manajemen data, dan kolaborasi tim.

## 📌 Fitur Utama
- **Manajemen Data Kegiatan, Keuangan, Proposal**: Input, edit, dan hapus data dengan mudah.
- **Autentikasi Pengguna**: Login dan registrasi dengan sistem keamanan dasar.
- **Dashboard Interaktif**: Tampilan ringkas untuk memantau aktivitas.
- **Integrasi GitHub**: Source code terkelola dengan version control.

## 🛠️ Teknologi yang Digunakan
- **Frontend**: HTML, CSS, JavaScript
- **Backend**: PHP (XAMPP environment)
- **Database**: MySQL
- **Version Control**: Git & GitHub

## 🚀 Cara Menjalankan Project
1. Clone repository:
   ```bash
   git clone https://github.com/ikomangagusadimerta/SIMKOM.git

2. Pindah ke folder project:
cd SIMKOM

3. Jalankan di XAMPP (htdocs).

4. Import database simkom.sql ke MySQL.

5. Akses melalui browser: http://localhost/rpl/

📂 Struktur Folder
/assets → file CSS, JS, dan gambar

/includes → file PHP pendukung

/database → file SQL

/index.php → halaman utama
